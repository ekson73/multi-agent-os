# DEPRECATED

This directory is deprecated as of v1.0.0.

## Migration

Hook scripts are now in `plugin-scripts/` directory:

- `plugin-scripts/session-start.sh`
- `plugin-scripts/pre-delegate.sh`
- `plugin-scripts/post-delegate.sh`
- `plugin-scripts/session-end.sh`

The `scripts/` directory was renamed to `plugin-scripts/` to avoid confusion with other script types and follow plugin conventions.

## Scheduled Removal

This directory will be removed in v2.0.0.
